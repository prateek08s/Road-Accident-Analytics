    ROAD ACCIDENT ANALYSIS

    The objective of this project is to perform data exploration using SQL on a dataset that contains information about countrywide traffic accidents in the UK from 2019 to 2022. The dataset was initially stored in SQL Server for analysis purposes. Once the analysis was completed, the dataset was loaded into Tableau, where it was visualized through informative and visually appealing charts and graphs.

    ROAD ACCIDENT DASHBOARD

    A dynamic and interactive dashboard was created through a process involving requirements gathering, data cleaning, processing, and visualization, providing valuable insights on total accidents, casualties by location, vehicle type, weather conditions, and tracking key performance indicators like year-over-year growth, enabling informed decision-making for enhancing road safety.

    ROAD ACCIDENT STORY

    The captivating Tableau story employs diverse visualizations like donut charts, bar charts, bubble charts, and maps to present road accident information. Donut charts depict casualty distribution by weather conditions and road surfaces, while bar charts highlight casualties by road types. Maps visualize accident severity and geographical distribution of casualties. This comprehensive combination of visualizations enables viewers to gain valuable insights and make informed decisions for enhancing road safety.
